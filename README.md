# Customize Nukes Window Title 🪟

A custom EventFilter is used to catch if the window title is changing (script name changes or script is modified) 🚀
This is produces in comparsion to the builtin callbacks (OnScriptLoad, OnScriptSave, OnUserCreate, KnobChanged) much less overhead and 
reduces the the update of the window title to a minimum.
